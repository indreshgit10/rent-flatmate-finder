# Rent and Flatmate Finder - Server

Backend API for the Rent and Flatmate Finder platform built with Node.js, Express, and MongoDB. The platform lets room owners post listings and tenants create preference profiles. An AI-powered compatibility engine (Gemini API with a rule-based fallback) scores and ranks tenant-listing matches. Accepted interest requests unlock real-time chat via Socket.io, with email notifications sent on key events.

## Setup

```bash
cp .env.example .env  
npm install
npm run dev
```
